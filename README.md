# Data Story Reports

Turn raw data into clear insights and decision-ready reports through narrative-driven HTML presentations. Designed to replace cluttered dashboards with structured, story-first data experiences.

---

## ✨ What is this?

**Data Story Reports** is a skill specification for generating:

- Narrative-driven data reports
- Clean, structured HTML presentations
- Insight-first analysis (not chart overload)

Instead of dashboards full of charts, this approach focuses on:
👉 clarity  
👉 storytelling  
👉 decision-making  

---

## 🚀 Key Features

- 📊 Transform raw data into insights
- 🧠 Story-first report structure
- 🎯 Decision-ready outputs
- 🌐 Single-file HTML (zero dependencies)
- 🎨 Clean, intentional design (no "AI slop")

---

## 🧩 Use Cases

- Business reports
- Product analytics summaries
- Executive briefings
- Data storytelling presentations

---

## 🏗️ Structure

The report follows a clear narrative flow:

1. Title (context + key takeaway)
2. Key Insights (3–5 findings)
3. Overview (KPIs + trends)
4. Deep Dive (question → chart → insight)
5. Conclusion (actions)
